# Kulturehire-Generation-Z-Aspirations-
This research project investigates the interests, aspirations, motivations, and preferences of individuals born between 1995 – 2012, a generation referred to as Gen Z.
The project focused on three major criteria which are:

 Learning Aspirations
 
 Work Aspirations
 
 Employer Aspirations
 
By capturing perspectives from over 78,000 participants, the project seeks to help organizations, educators, and policymakers better align with Gen Z’s values, expectations, and career planning frameworks.

# Data Collection
The data used in the study was collected via survey, implementing the usage of online form for the data collection. Various Question formats such as Multiple Choice, Likert, Rating and Closed Response were implemented in the survey.


# Participants
Responses from 78,552 participants were considered in the study including both gender (male and female). 


# Data Analysis
Quantitative analysis of trends.

Gender-based segmentation

Data visualization using various charts to identify sentiment, preferences, and behaviour patterns.

# ANALYSIS TOOLS & USAGE

# Microsoft Excel for Data Cleaning 

Microsoft Excel played a vital role in the initial stages of the data handling process. It was used to clean raw survey data collected from over 78,000 participants. Power Query Editor and Functions like Text to Columns, Remove Duplicates, and Data Validation were used to standardize and structure the dataset. 

# MySQL for Data Storage & Analysis

MySQL was used to manage and query the structured dataset more effectively. After cleaning the data in Excel, it was loaded into MySQL Workbench. Here, segmentation of the data—especially for gender-based analysis and trend identification was done which enabled deeper insights through logical filtering and grouping.

# Power BI for Advanced Visualization

Microsoft Power BI was employed for in-depth data storytelling and interactive visualization with usage of various charts and slicers. It allowed the creation of dynamic dashboards displaying KPIs.
